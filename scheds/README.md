SCHED_EXT SCHEDULERS
====================

# Introduction

This directory contains the repo's schedulers.

These schedulers are simply examples of different types of schedulers that
can be built using `sched_ext`. They can be loaded and used to schedule on
your system, but their primary purpose is to illustrate how various 
features of `sched_ext` can be used.

Please see the following `README` files for details on each of the various types
of schedulers:

- [`c`](c/README.md) describes all of the schedulers with `C` user space
  components.
